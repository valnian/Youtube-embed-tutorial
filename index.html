How to Embed a Video on Your Website

A step-by-step tutorial with screenshots to help you easily add videos to your web pages

Why Embed Videos?

Embedding videos on your website can increase engagement, improve SEO, and make your content more dynamic. Most video hosting services like YouTube, Vimeo, and Dailymotion provide easy ways to embed their content on your site.

In this tutorial, we'll focus on embedding YouTube videos, which is one of the most popular video platforms.

1

Find the Video

Navigate to the video you want to embed

Go to YouTube and find the video you'd like to embed on your website. Make sure the video has embedding allowed (most do).

YouTube homepage - find your video
2

Click Share

Locate and click the Share button

Below the video player, you'll see a Share button. Click on it to reveal the sharing options.

Click the Share button below the video
3

Select Embed

Choose the Embed option from the menu

From the sharing options that appear, select "Embed" to generate the embed code for the video.

Select the Embed option from the menu
4

Copy the Code

Copy the provided embed code

YouTube will generate an HTML iframe code for embedding the video. You can customize the player options if needed, then click "Copy" to copy the code to your clipboard.

Copy the generated embed code
5

Paste in HTML

Paste the code into your website's HTML

Paste the copied code into your website's HTML file where you want the video to appear. The code will look something like this:

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>
Paste the code in your HTML file
